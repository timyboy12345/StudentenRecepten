# StudentenRecepten
This is the Repo that serves [studentenrecepten.com](https://studentenrecepten.com).


## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
